# Open-GDR Frontend

The frontend interface for the Open-GDR project, built with **React**, **TypeScript**, **Vite**, and **TailwindCSS**.

For a detailed architecture and component guide, please refer to [FRONTEND_GUIDE.md](../FRONTEND_GUIDE.md).

## Quick Start
```bash
npm install
npm run dev
```

## Key Fetaures
- **Timeline UI**: Stream of consciousness notes.
- **Agentic Chat**: Chat with PDF documents via RAG.
- **Voice Mode**: Real-time voice interaction with the system.
- **Rolling Summary**: Daily AI-generated briefings.
